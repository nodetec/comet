#teaching #science/astronomy #maps

First lesson for beginners: every star map is an argument about purpose.
A navigation chart values precision over beauty.
A public atlas values recognition over density.
A teaching map should probably value courage, because new readers stop trusting themselves long before the stars become the hard part.
